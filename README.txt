Панель управления или что-то еще например "HTTP Клиент" пока не определено. Возможно стоит сделать что-то универсальное. Название предварительное ModCot(производное от двух слов но каких уже забыл:))).
Задумывалось для обучения и работе с WordPress REST API.
Позволяет как получать данные с сайта так и в будущем изменять, обновлять и создовать.

***********
По страницам принято такое соглашение что по каждой странице идет отдельный файл js.
Страница "Последние записи" - выдает  данные с сайта по записям на данный момент пока 5-ть записей выводится на одной странице. Можно нажав на определенную запись перейти к ее тексту.
Страница "Категории" - выдает категори сайта только пока не все а только 10-ть, необходимо разработать механизм вывода всех категории по странично.
Страница "Настройки" - пока только сохранияет адрес сайта с которого идет скачивание данных.

***********
По модулям принято соглашение что каждый модуль который используется многократно и на разных страницах записывается в файл modules.js.
Модуль пагинации: перенесен в файл modules.js. Продумать механизм увеличения - когда заканчиваются записи, либо вывод ошибки либо ничего не выводить.
